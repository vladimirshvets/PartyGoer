# PartyGoer
A tiny telegram bot.


##Setup:
- C# 11, .NET 7.0
- MSSQL Server 2022 (Docker container on Apple M1/M2 Silicon)
  * Note: Docker Desktop Dashboard -> Settings -> “Features in development” -> select the “Use Rosetta for x86/amd64 emulation on Apple Silicon” checkbox.
